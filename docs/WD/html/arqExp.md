```html
<!-- 
    => DOCTYPE indica o tipo de formato do arquivo a ser interpretado pelo navegador. No caso abaixo é "html"
    => No VS Code, graças ao Emmet, basta digitar uma exclamação (!) para criar uma base de código HTML.
-->
<!DOCTYPE html>
<!-- 
    => A tag html indica a raiz de toda página html.
    O atributo lang na tag html indica a linguagem na qual o conteúdo da página está escrito. 
    => Para saber mais acesse: https://tableless.com.br/declarando-idiomas-no-html/
-->
<html lang="pt-br">
    <!-- 
        O head irá conter as configurações da página html.
        É parte "invisível" do site.
        Mais sobre a head: https://www.w3schools.com/html/html_head.asp 
    -->
    <head>
        <!-- tags html -->
    </head>

    <!-- O body contem a parte "visível" do site -->
    <body>
        <!-- tags html -->
    </body>
</html>
```
